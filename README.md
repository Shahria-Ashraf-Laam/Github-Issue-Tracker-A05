1️⃣ What is the difference between var, let, and const?
 var => old JavaScript variable, function scoped, can be re-declared <br>

 let => block scoped, value can be changed but can't be re-declared in the same scope <br>

 const => block scoped, once declared the value can't be changed <br>

2️⃣ What is the spread operator (...)?
 We use spread operator (...) in JavaScript to expand/spread arrays, strings, or objects <br>

<hr>

3️⃣ What is the difference between map(), filter(), and forEach()?
 map() => Copies each element from an array and returns a new array <br>

 filter() => Selects elements based on a condition and returns new array <br>

 forEach() => Traverses the whole array/object elements one by one <br>

<hr>

4️⃣ What is an arrow function?
 Arrow function in JavaScript is a shorter way to write function expressions, introduced in ES6. ( () => {} )  <br>

5️⃣ What are template literals?
 Template literals in JavaScript are string literals that allow multi-line strings introduced in ES6. <br>
